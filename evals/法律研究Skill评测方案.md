## 一、评测目标

本方案评测「法律研究」Skill 在处理中国法律问题时的输出质量，聚焦三个核心指标：

| 指标 | 定义 | 计算方式 |
|------|------|---------|
| **结论准确性** | 核心法律结论是否与参考答案一致 | 人工逐条判定，每条结论 pass/fail |
| **法条召回率** | 参考答案中的关键法条，Skill 是否全部引用 | 实际引用的关键法条数 ÷ 参考答案关键法条总数 |
| **法条精准率** | Skill 引用的法条中，条号正确、原文无误、现行有效、适用于本问题的占比 | 引用准确的法条数 ÷ Skill 引用的法条总数 |

> **注**：法条精准率惩罚两类错误：①引用了不相关或错误的法条；②条号或原文有误。

## 三、各题参考答案与评测标准

### eval-1：建设工程施工合同未备案效力

**研究问题**：建设工程施工合同未办理备案手续，是否影响合同效力？

**参考答案核心结论**：
1. 备案属"管理性规范"而非"效力性强制规定"，未备案不影响合同效力
2. 强制备案制度已被正式删除（2018年9月《修改决定》、2019年3月《修改通知》）
3. 最高人民法院多份判决确认未备案合同有效

**参考答案关键法条**（召回率分母）：
- 《招标投标法》第47条
- 《房屋建筑和市政基础设施工程施工招标投标管理办法》原第47条（已删除）
- 《建设工程施工合同纠纷司法解释（一）》或（二）相关条款

**结论评测标准**：

| 结论项 | 判定标准 |
|--------|---------|
| 未备案合同有效 | 备忘录明确得出"未备案合同有效"或等效表述 |
| 区分管理性规范与效力性强制规定 | 文中出现该区分或等效论述 |
| 识别备案制度已被废除 | 文中提及2018/2019年修订删除备案要求 |

### eval-2：工伤复发停工留薪期（北京）

**研究问题**：工伤职工工伤复发后再次休假，甲公司是否应给予停工留薪期待遇？

**参考答案核心结论**：
1. 工伤复发认定：北京实操中经办机构通常不受理核准申请，实践以医院诊断证明为主要依据
2. 诊断证明载明"工伤复发"的，裁审机关普遍认可；未载明但伤情与原工伤部位一致的，亦可作为依据
3. 停工留薪期依据《北京市停工留薪期分类目录》确定，一般不超过12个月
4. 停工留薪期内原工资福利待遇不变

**参考答案关键法条**（召回率分母）：
- 《工伤保险条例》第33条（停工留薪期一般不超过12个月，原待遇不变）
- 《工伤保险条例》第38条（工伤复发享有停工留薪期待遇）
- 《北京市工伤职工停工留薪期管理办法》第3条、第11条

**结论评测标准**：

| 结论项 | 判定标准 |
|--------|---------|
| 工伤复发可获停工留薪待遇 | 备忘录结论明确甲公司应给予停工留薪待遇 |
| 停工留薪期不超过12个月 | 文中出现"12个月"或"一年"上限说明 |
| 北京地方规定适用 | 引用《北京市工伤职工停工留薪期管理办法》或《北京市停工留薪期分类目录》 |

### eval-3：简易注销后保证担保责任

**研究问题**：有限责任公司甲公司在保证担保存续期间内，由股东张某办理简易注销，董事为李某；甲公司注销后，债权人乙公司可向哪些主体主张权利、主张何种责任？

**参考答案核心结论**：
1. 保证债务具有独立性，不因保证人（甲公司）注销而当然消灭
2. 股东张某因承诺不实，依《公司法》第240条承担连带责任，不以出资额为限
3. 股东可依《民法典》第700条向原债务人追偿

**参考答案关键法条**（召回率分母）：
- 《公司法》（2023年修订）第240条
- 《公司法司法解释二》第20条
- 《民法典》第681条（保证合同定义）
- 《民法典》第700条（保证人追偿权）

**结论评测标准**：

| 结论项 | 判定标准 |
|--------|---------|
| 保证责任不因注销消灭 | 文中明确说明保证债务独立性，注销不消灭责任 |
| 股东承担连带责任 | 备忘录明确张某须承担连带责任 |
| 责任不以出资额为限 | 文中出现该表述或等效说明 |
| 股东享有追偿权 | 文中提及张某可向主债务人追偿 |

### eval-4：短线交易认定

**研究问题**：张某在某上市公司工作，不属于董事、监事、高管或大股东。2025年9月以协议转让方式买入20万股，2025年9月30日签订协议，2025年10月25日完成过户登记。2026年2月经董事会任命为副总经理（高管）。2026年4月20日卖出上述股票。请问张某是否构成短线交易？

**参考答案核心结论**：
1. 适用"一端说"原则：买入时不具备特定身份，卖出时具备的，应适用短线交易规定
2. 时点认定以"证券过户登记日"为准：买入时点为2025年10月25日（过户登记日），卖出时点为2026年4月20日，间隔不足6个月
3. 张某卖出时已具备高管身份，构成短线交易

**参考答案关键法条**（召回率分母）：
- 《证券法》第44条（短线交易归入权）
- 证监会关于短线交易认定的相关规定或指引（如适用）

**结论评测标准**：

| 结论项 | 判定标准 |
|--------|---------|
| 构成短线交易 | 备忘录结论明确张某构成短线交易 |
| 适用"一端说"（卖出时具备身份即适用） | 文中论述买入时无需具备身份，卖出时具备即可 |
| 时点以过户登记日为准 | 文中明确采用过户登记日而非协议签订日计算六个月 |
| 法律后果：收益归入公司 | 文中说明短线交易收益归上市公司所有 |

## 四、评分方法

### 4.1 结论准确性

对每题的结论项逐条人工判定 `pass` / `fail`：

```
单题结论准确性得分 = 通过结论项数 / 该题结论项总数
```

### 4.2 法条召回率

```
单题召回率 = Skill 引用的关键法条中与参考答案匹配的数量 / 参考答案关键法条总数
```

匹配标准：法规名称和条号与参考答案一致（不要求原文逐字相同）。

### 4.3 法条精准率

```
单题精准率 = 引用准确的法条数 / Skill 引用的法条总数
```

以下任一情况判定为"不准确"：
- 条号有误（如引用了不存在的条款）
- 法规已失效但未标注
- 原文内容与实际法条明显不符
- 法条与本题无关

### 4.4 整体评级

| 三项指标平均分 | 评级 |
|----------------|------|
| ≥ 90% | A — 优秀 |
| 75%–89% | B — 良好 |
| 60%–74% | C — 合格 |
| < 60% | D — 需改进 |

## 六、注意事项

- 参考答案不一定穷举所有正确结论，Skill 输出若包含额外合理论证，不扣分
- 断言判定以实质内容为准，不要求文字完全一致
- 若参考答案与法律原文存在出入，以法律原文（元典 API 核实结果）为准，参考答案降级为"实务观点参考"
- eval-2 存在北京地方特殊性，结论中须体现地域限定
