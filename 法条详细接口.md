
## 1 法条详情接口

接口描述：根据法规ID、法规名称、法条号和时间，检索法条内容

分类：法律法规请求类型：POST调用类型：普通HTTP

调用地址：
https://apiplatform.legalmind.cn/open/rh_ft_detail

请求参数：

| 字段名          | 类型     | 是否必填 | 说明                                               |
| ------------ | ------ | ---- | ------------------------------------------------ |
| `id`         | string | 否    | 详情目标 ID。                                         |
| `fgmc`       | string | 否    | 法规名称（当 `id` 为空时必填）                               |
| `ftnum`      | string | 否    | 法条号/名称（当 `id` 为空时必填）                             |
| `refer_date` | string | 否    | 参考日期（用于确定当时生效的法律法规，如未提供，则为当前生效的版本，格式为yyyy-MM-dd） |
|              |        |      |                                                  |

- **body 为空 JSON**：返回失败 `message = "请求参数不能为空"`
- **`id` 与 `fgmc` 和 `ftnum` 同时为空**：返回失败（501）`message = "id与法规名称不可同时为空！"


通用返回结构（ResultVO）

|字段名|类型|说明|
|---|---|---|
|`status`|string|`success` / `failed` 等|
|`code`|number|成功通常为 200；失败可能为 500/501 等|
|`message`|string|提示信息|
|`data`|object \| null|详情对象；异常时可能为 null|

法条详情对象字段

| 字段名       | 类型     | 说明                                                     |
| --------- | ------ | ------------------------------------------------------ |
| `type`    | string | 固定为 `法条`                                               |
| `id`      | string | 法条文档 ID                                                |
| `fgid`    | string | 所属法规 ID                                                |
| `tid`     | string | 条编号，例如：100                                             |
| `url`     | string | 详情地址：`/zxt/statuteDetail/detailPage/{fgid}?text={tid}` |
| `ftmc`    | string | 法条名称，例如：中华人民共和国刑法(2023修正)第一百条                          |
| `title`   | string | 同 `ftmc`                                               |
| `ft_num`  | string | 法条号/名称，例如：第一百条                                         |
| `fgmc`    | string | 法规名称                                                   |
| `content` | string | 法条内容                                                   |
| `sxx`     | string | 所属法规时效性                                                |
| `xljb_1`  | string | 所属法规效力级别-一级                                            |
| `xljb_2`  | string | 所属法规效力级别-二级                                            |
| `fbrq`    | string | 所属法规发布日期                                               |
| `ssrq`    | string | 所属法规实施日期                                               |

请求示例：

curl -X POST "https://apiplatform.legalmind.cn/open/rh_ft_detail" -H "accept: application/json;charset=UTF-8" -H "Content-Type: application/json" -d "{ \"fgmc\": \"中华人民共和国刑法\", \"ftnum\": \"第一百条\", \"refer_date\": \"2025-01-01\"}"
