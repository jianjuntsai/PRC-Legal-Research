# 法律分析：股东未实缴出资时债权人能否追索股东

## 结论

可以。公司股东未实缴出资，公司无法清偿债务时，债权人可以追索股东，要求其在未实缴出资范围内承担补充赔偿责任。

## 分析

### 1. 法律依据

根据《最高人民法院关于适用公司法若干问题的规定（三）》第13条，公司债权人可以请求未履行出资义务的股东在未出资本息范围内对公司债务不能清偿的部分承担补充赔偿责任。

根据《公司法》（2023修订）第54条，公司不能清偿到期债务的，已到期债权的债权人有权要求已认缴出资但未届出资期限的股东提前缴纳出资（即"出资加速到期"制度）。

### 2. 责任性质

股东承担的是**补充赔偿责任**，而非连带责任：
- 以公司财产不足清偿部分为限
- 以股东未实缴出资本息为上限

### 3. 认缴期限问题

- 认缴期限已届满：可直接依公司法解释三第13条追索
- 认缴期限未届满：依公司法第54条，债权人可要求加速缴纳

## 风险提示

- 各地法院对"不能清偿到期债务"认定标准不一
- 加速到期是否需要先经过诉讼程序存在争议
- 多名债权人竞合时存在先行权问题
