# 法律分析：未约定补偿金的竞业限制条款效力

## 结论

该竞业限制条款**有效**，两年期限合规，但公司须向员工支付补偿金。

## 分析

### 1. 条款效力

根据《劳动合同法》第23条，竞业限制条款须约定按月补偿。未约定补偿金不直接导致条款无效，但公司须补足补偿（月均工资的30%）。

### 2. 期限合规

《劳动合同法》第24条规定竞业限制期限不得超过两年，本案正好为两年，合规。

### 3. 适用对象

须确认员工属于高级管理人员、高级技术人员或其他负有保密义务的人员，否则条款无效。

### 4. 员工权利

若公司3个月未支付补偿金，员工有权申请解除竞业限制约定。

## 风险

- 各地法院裁判尺度不同，少数法院在综合考量后可能认定无补偿的竞业限制条款无效
- 若员工不属于法定适用对象，条款无效
