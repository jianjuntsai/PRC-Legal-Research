# 法律分析：中小企业间背对背付款条款的效力

## 结论

双方均为中小企业时，"收到业主付款后再付分包款"的背对背条款原则上有效，但需注意以下风险。

## 分析

### 1. 大型企业情形

根据最高院2024年批复，大型企业与中小企业约定背对背条款无效。但你们双方均为中小企业，该批复不适用。

### 2. 中小企业之间

目前法律无明文禁止中小企业之间约定背对背条款，依合同自由原则，该条款有效。

### 3. 可能被突破的情形

- 业主已进入破产程序，付款条件不可能成就
- 条款约定不明（如"参考业主回款比例"）
- 总包方未积极向业主催款

## 建议

- 明确约定付款触发条件和最长等待期限
- 保存催款记录
- 约定业主无力付款时的应急安排
