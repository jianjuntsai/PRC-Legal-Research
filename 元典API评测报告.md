## 三、API调用与检索结果

### 3.1 企业合规领域

**调用方式**：
```python
search_fatiao("涉案企业合规 不起诉", sxx="现行有效", top_k=8)
search_fagui("涉案企业合规 不起诉", sxx="现行有效", top_k=5)
search_ptal(qw="企业合规整改 不起诉", ajlb="刑事案件", top_k=5)
search_qwal(qw="国有企业 合规 不起诉", top_k=5)
```

**核心命中结果**：

| 检索类型 | 命中数 | 关键结果 |
|---------|-------|---------|
| 法条 | 8条 | 《第三方监督评估指导意见》第2、14条；《知识产权案件工作指引》第17条 |
| 法规 | 5条 | 全部为现行有效；命中2021年指导意见+2022年办法（司法解释） |
| 普通案例 | 49条 | 合规整改不起诉案例，主要为民营/小微企业 |
| 权威案例 | 1条 | **新泰市6家建筑企业串通投标案（含2家国有企业）** |

**关键发现**：

> 《第三方监督评估指导意见》第6条明确规定"**国务院国有资产监督管理委员会、财政部负责承担管委会中涉及国有企业的日常工作**"，制度设计上已为国有企业预留适用接口。第5条排除情形亦未将国有企业列为例外。

> 最高检第四批典型案例（新泰市案）中，2家国有企业因被胁迫参与串通投标，检察机关对其适用合规不起诉，是目前可检索到的最具代表性的国有企业适用先例。

### 4.3 优势

1. **核心法条精确命中**：`get_fatiao_detail`直接获取完整条文，准确度100%。
2. **权威案例质量高**：三个领域均覆盖最高院/最高检发布的典型案例，内含完整裁判要旨。
